# 게임 설명

### 📱✨ **워드이터 (Word Eater)**

**버려진 스마트폰 속**, 데이터 더미에서 탄생한 작은 네트워크 생명체 **워드이터**!

너만의 워드이터를 발견하고, 모으고, 키워 **성체로 진화시키는 초단기 육성 게임**!

단어 조합, 추리, 미니게임을 통해 올바른 데이터를 먹여 성장시키고,

귀여운 생명체가 성체가 되어 **도감에 기록되는 순간의 짜릿함**을 경험해보세요!

## 🧩 **게임 콘셉트 & 핵심 특징**

- **📌 콘셉트**
    
    버려진 스마트폰 속 폴더에서 기어나오는 미스터리한 생명체 **워드이터**
    
    이들에게 데이터를 먹이고 성장시키며 **성체로 완성**하는 짧고 몰입감 있는 육성 게임!
    
- **💡 핵심 재미 요소**
    - 머리 식히는 **가벼운 단어 추리 플레이**
    - 짧고 강렬한 **육성 루프**
    - 귀여운 네트워크 생명체를 길들이는 **수집욕 자극**
- **✨ 차별화 포인트**
    - 파이썬 기반의 **무한 확장 문제 생성**, 지속 가능한 업데이트
    - 단순 단어 게임을 넘어 **데이터 해석 + 힌트 추리 + 미니게임**의 3단 플레이
    - 3단 진화 시스템으로 매번 다른 성장 경험 제공


## 📚 **스토리 & 세계관**

- **📱 배경 설정**
    
    스마트폰 내부 세계를 재현!
    
    앱 아이콘, 키보드, 폴더 등 친숙한 오브젝트가 게임 내 UI/스테이지로 변환됩니다.
    
- **📖 주요 스토리**
    - 스마트폰 폴더 속에서 **워드 이터가 불시에 등장**
    - 데이터 더미 속에서 **랜덤 키워드를 감지**
    - 플레이어는 데이터를 올바르게 주입하여
        
        **비트 → 바이트 → 워드**, 3단계 진화를 완성
        
    - 결국 **성체 도감**에 등록하는 것이 목표!
- **🧸 주요 캐릭터**
    - **플레이어:** 워드이터를 보살피는 “관리자”
    - **워드이터:** 네트워크 속에서 떠돌다가 나타나는 생명체들

## 🎮 **게임 플레이 및 시스템**

### 🔋 **1. 성장 시스템 — 워드이터의 3단 진화**

- **Bit → Byte → Word**
- 단계별로 제공되는 **턴(배터리)**
    - 비트: 4턴
    - 바이트: 8턴
    - 워드: 12턴
- 턴 내에 올바른 데이터를 충분히 주입하지 못하면 **사망**

턴마다 다음 3가지 행동 중 선택 가능:


### 🍽 **2. 데이터 주입 (Feeding)**

- 클릭 시 **키보드 UI**가 등장
- 글자를 드래그해 조합하여 워드이터에게 ‘데이터 먹이’로 전달
- 단계별 올바른 조합 횟수
    - 비트: 2회 성공
    - 바이트: 4회 성공
    - 워드: 6회 성공
- 일정 이상 실패하면 사망


### 🧠 **3. 최적 알고리즘 (힌트 시스템)**

미니게임을 통해 **단어 힌트 데이터** 획득!

- **전화 형식**: 유사한 단어를 통째로 제공
- **메시지 형식**: 단어의 유사도 힌트 제공


### 🧹 **4. 노이즈 제거 (2턴 소모)**

난이도에 따라 다른 보상:

- **이지 모드:** 클리어한 미니게임 횟수 → 랜덤 글자 조각 지급
- **하드 모드:** 글자를 직접 선택해 가져갈 수 있는 단어 선택창 제공

추상도 높은 개념까지 확장되는 구조


### 🧬 **5. 워드이터의 특징**

- 경험을 통해 유저가 감각적으로 학습하도록 설계
- 총 20종 내외의 개성 있는 외형 제공 예정
- 워드이터는 진화하면 데이터가 초기화
- 그러나 이전 단계의 단어와 관련되어있는 더 고차원적인 단어로 변경

ex)

- “주사위 → 확률 → 통계학”
- “거울 → 반사 → 광학”

> 모든 단어 데이터, 힌트, 문장 힌트는 파이썬으로 자동 생성!


### ✔ 진화 성공 시

- 성체가 **도감에 영구 등록**

### ❌ 사망 시

- 선택지가 등장
    - 정답 확인
    - 그냥 포기
    - 광고를 보고 <부활> 가능


### 📗 **6. 도감 시스템**

- 키운 워드 이터의 기록·성체 모습·정답 단어까지 저장
- 진화 과정 비주얼도 확인 가능
- 실제 스마트폰 갤러리를 오마주한 UI
